## 安装引用
```shell
npm install 或 yarn
```
## 使用说明
小程序无法使用本地资源，首先开启本地静态资源服务，发布的时候再改为正式的静态服务地址；
```shell
npm run assetsServe
npm run dev:weapp # 或者 npm run dev:h5
```
